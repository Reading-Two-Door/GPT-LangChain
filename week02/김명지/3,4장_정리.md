# 3장. 파이썬 개발환경 준비

## 3-1. 파이썬 개요
### 파이썬이란?
- 단순, 범용성 높은 프로그래밍 언어
- 다양한 인공지능 개발용 라이브러리 사용가능
  
  - 파이토치: 페이스북개발, 동적계산그래프, 파이썬네이티브 지원, CUDA지원, 연구분야선호.  
    - 동적계산그래프: Define-by-run, 데이터를 흘려보내면서 계산그래프를 만들어간다. 2.0부터는 텐서플로우도 지원.
    - 파이썬 네이티브: 파이썬으로 작성된 이라는 의미.
    - CUDA: GPU를 사용하여 병렬 코드를 실행하는 데 사용되는 라이브러리 NVDIA 개발.
  
  - 텐서플로우: 구글개발, kerasAPI(모델생성쉬움), 엣지디바이스지원강화, 산업분야선호.  
    - keras API: API 호출을 통해 모델을 생성하고 학습시킬수 있는 기능을 제공하는 API.
    - 엣지디바이스: 소비자에게 가까운 위치에 있는 디바이스를 의미합니다. 이는 개인용 노트북, 휴대전화, 스마트 홈 가전제품 등.
